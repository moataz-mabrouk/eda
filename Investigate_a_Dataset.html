Data overview¶

The following columns will be dropped as they will non be used in the analysis:

  • production_companies
  • release_date
  • keywords
  • tagline
  • homepage

# Create a data frame with genre count for each year
year_list= sorted(df_genres_clean['release_year'].drop_duplicates().tolist())
genrs_year_df = pd.DataFrame()
for year in year_list:
    genre_dict = {}
    genres_in_year = df_genres_clean[df_genres_clean.release_year == year]
    genres_in_year = genres_in_year["genres"].values
    for cell in genres_in_year:
        genres_row = cell.split("|")
        for genre in genres_row:
            if genre not in genre_dict:
                genre_dict[genre] = 1
            else:
                genre_dict[genre] = genre_dict[genre] + 1
                    
    df_row = pd.DataFrame(genre_dict, index = [year])
    genrs_year_df = genrs_year_df.append(df_row, ignore_index = False, sort = True)
#replacing nan values with zero and 
genrs_year_df.fillna(0, inplace = True)

actor_dict = {}

actors = df_genres_clean["cast"]
#print(actors)
actors = actors.str.split("|")
#print(actors)
actors = np.array(actors)
#print(actors)
for actor_list in actors:
    for actor in actor_list:
        actor = actor.strip()
        if actor not in actor_dict:
            actor_dict[actor] = 1
        else:
            actor_dict[actor] += 1

Limitations¶

the Data is so vast and there is a lot of more information that can be extracted from it such as:

  • most active directors
  • how many times did an actor appear in movies each year
  • more correlation between revenue and other features
  • correlation between high votes and other features
and much more, however there is a lot of missing data which hinders our ability to use this data to its fullest potential

Note:¶

the original data frame was separated to two data frames to decrease the dropped missing data as much as possible
